# MAINTENANT — Dark Crystal

Version complète de l’interface avec une direction visuelle **Dark Crystal Social**.

## Effets intégrés dans le code

- cartes facettées et translucides ;
- contours cristallins verts et turquoise ;
- reflets prismatiques légers ;
- bouton central en forme de cristal ;
- navigation inférieure en verre ;
- panneaux, formulaires et fenêtres en verre sombre ;
- éclats de fond discrets ;
- contraste conservé pour la lisibilité.

## Installation

1. Décompressez le ZIP.
2. Remplacez tous les fichiers du projet Vercel actuel.
3. Redéployez.
4. Fermez puis rouvrez le navigateur sur le téléphone si l’ancien cache reste visible.

Les données du prototype restent stockées localement sur le téléphone.
